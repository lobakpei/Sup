England v1 • How it works

PaddleReady gives you live conditions for stand-up paddleboarding launch spots in England. It combines public weather, river level and sewage data into a simple recommendation.

Important disclaimer

This is informational only, like a weather forecast. It is not a safety guarantee. Conditions change quickly. You are responsible for your own assessment, your skills, equipment, and for holding a valid waterways licence where required. We accept no liability for decisions made using this information.

Data sources

  • Weather: Open-Meteo (free, accurate)
  • River levels: Environment Agency real-time API
  • Sewage advisory: Based on public storm overflow data
  • Parking: OpenStreetMap

How to add your own spots

Open the file in a text editor. Find the LAUNCH_POINTS array near the top.
Copy one of the example objects and change the values. Save and reload.

Recommended: Find the correct Environment Agency station reference for accurate river data.
